🧠 Challenge Description
-
Send the cyros a WhatsApp DM and get the reward, don’t forget to give him feedback…

The challenge clearly suggests:

Contacting “cyros” directly

Using WhatsApp

Sending feedback to receive the flag

🔎 Solution Approach
-
1️⃣ Identified the official contact/WhatsApp number associated with “cyros”.

2️⃣ Sent a WhatsApp DM as instructed.

3️⃣ Included a short feedback message.

4️⃣ Received an automated (or manual) reply containing the flag.

This challenge tests:

Misc skills

Social interaction

Following instructions carefully
