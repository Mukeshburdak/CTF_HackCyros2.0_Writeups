🧠 Challenge Description
-
This photo shows you something, but not everything. Find out what lies within.

Flag format: cyros{Name_of_Temple}

The image hinted at a temple structure, but the full identity was not directly provided.

🔎 Solution Approach
-
1️⃣ Carefully observed the architectural style in the image.

2️⃣ Noticed the stepwell structure going downward.

3️⃣ Performed a Google image search using visual clues.

4️⃣ Matched the structure to a famous historical stepwell temple in India.

The structure was identified as:

Rani ki Vav

🚩 Final Flag
-
cyros{Rani_ki_Vav}

🧩 Key Takeaway
-
Reverse image search is powerful in OSINT challenges.

Architectural patterns and landmarks can quickly reveal locations.

Always analyze visual clues before searching.
