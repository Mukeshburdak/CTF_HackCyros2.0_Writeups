# 🚩 CTF_HackCyros2.0_Writeups

Welcome to the **CTF_HackCyros2.0_Writeups** repository!

This repository contains detailed writeups and solutions for the challenges from **HackCyros 2.0**, a Capture The Flag (CTF) event organized by **GDGoC CU Jammu**.

---

## 📌 About the Event

HackCyros 2.0 was a cybersecurity-focused CTF competition designed to test participants' skills in various domains such as:

- 🛡️ Web Exploitation  
- 🔐 Cryptography  
- 🧠 Reverse Engineering  
- 🧪 Forensics  
- 🧩 Miscellaneous  

This repository provides structured explanations of the problems along with step-by-step solution approaches.

The goal is to:
- Help beginners understand how CTF challenges are solved
- Share learning resources with the community
- Document practical cybersecurity techniques

---


Each folder contains:

- 📄 Problem Description  
- 🛠️ Tools Used  
- 🔎 Analysis Process  
- 🧩 Step-by-Step Solution  
- 🏁 Final Flag  

---

## 🛠️ Tools & Technologies Used

Some commonly used tools in the writeups include:

- Wireshark  
- CyberChef  
- Burp Suite  
- Strings / Ghidra  
- Steghide  
- Binwalk  
- Basic Linux commands  
- Python scripts (where applicable)

## 🚀 How to Use This Repository

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Mukeshburdak/CTF_HackCyros2.0_Writeups.git
```

### 2️⃣ Navigate into the Directory

```bash
cd CTF_HackCyros2.0_Writeups
```

### 3️⃣ Open Any Challenge Writeup

Browse into the respective category folder and open the `.md` file to read the solution.
